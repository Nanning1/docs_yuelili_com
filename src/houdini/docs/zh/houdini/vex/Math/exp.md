## 描述

Returns the exponential function of the argument.

```c
float  exp(float n)
```

```c
<vector> exp(<vector>n)
```

Returns the [Exponential function
\_\_](http://en.wikipedia.org/wiki/Exponential_function)of`n`. For vectors, this
is done per-component.

返回 n 的指数函数。对于向量，这是按分量进行的。
