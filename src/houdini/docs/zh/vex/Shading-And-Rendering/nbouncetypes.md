# nbouncetypes

VEX function

#

```c
int  nbouncetypes()
```

Return the number of different bounce types supported by the renderer.

返回渲染器支持的不同弹跳类型的数量。

See [bouncemask](bouncemask.html) for information on component label bitmasks.

关于组件标签位掩码的信息，请参见 ebouncemask。
