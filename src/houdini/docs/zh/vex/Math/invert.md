## 描述

Inverts a matrix.

```c
<matrix> invert(<matrix>m)
```

`m`

The matrix to [invert \_\_](http://en.wikipedia.org/wiki/Invertible_matrix).

要反转的矩阵。

Returns

The inverted matrix, or the original matrix if the matrix is singular.

被反转的矩阵，如果矩阵是单数，则是原始矩阵。
