## 描述

Returns the squared distance of the vector or vector4.

```c
float  length2(vector2 v)
```

```c
float  length2(vector v)
```

```c
float  length2(vector4 v)
```

Returns the squared distance of the vector.

返回矢量的平方距离。

## Examples

    length2({0.5, 0.75, 0}) == 0.8125;
